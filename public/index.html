📋 How it works

  1. Select Profile — Choose which browser session to use
  2. Click Record — A remote browser window opens in the viewer below
  3. Do your task — Click, type, navigate inside the viewer — all recorded automatically
  4. Popups auto-dismissed — They won't be recorded as steps
  5. Stop Recording — Edit/remove unwanted steps
  6. Save as Flow — Run this flow anytime, it replays exactly!

🍪 Session Persistence

Login once during recording — your session cookies are saved. Next time the flow runs, it skips login automatically!

👥 Multiple Accounts

Use different profiles for different accounts. Each profile has its own saved session.
